This post is a **draft** — it never appears on the live site while `draft: true` is set in the header above.

To publish a real post:

1. Copy this file and rename it — the file name becomes the URL, so `why-ai-fits-trades-businesses.md` publishes at `/blog/why-ai-fits-trades-businesses`.
2. Fill in the `title`, `description` (used in Google results and the blog index), and `date`.
3. Write the post below the `---` line in plain Markdown.
4. Set `draft: false`.

The blog index, RSS feed, and sitemap all update automatically. The Blog link appears in the site nav as soon as the first published post exists.

## Writing style reminders

Plain, human, NZ-English tone. No em dashes. No prices. Short and specific beats long and general — the goal is answering one real question a small-business owner would actually search for.
